On my Debian and LMDE system, this adapter (USB ID: 2357:0604) would not work automatically, I had to upgrade to a newer kernel first (this should be available from Linux 5.15 onwards) by enabling the backports, adding this line in /etc/apt/sources.list:

deb http://deb.debian.org/debian bullseye-backports main non-free contrib

and then installing the new kernel:

apt update
apt install linux-image-6.1.0-0.deb11.5-amd64-unsigned

unfortunately this was not enough, even after rebooting with the new kernel I had a problem with the firmware missing:

[...] firmware load for rtl_bt/rtl8761bu_fw.bin failed with error -2
firmware file rtl_bt/rtl8761bu_fw.bin not found

The firmware is available from the kernel git repo, you need both rtl8761bu_config.bin and rtl8761bu_fw.bin save them in /lib/firmware/rtl_bt/ and remove and reinsert the USB BT adapter.

Hope this helps someone out there.
