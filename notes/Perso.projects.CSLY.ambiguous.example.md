# simple ambiguous grammar.

## grammar
```
<S>
⇒ <S> <S>
⇒ (<S> <S>) <S>
⇒ ('a' <S>) <S>
⇒ ('a' 'a') <S>
⇒ ('a' 'a') 'a'
⇒ "aaa"
```
